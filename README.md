# Portafolio

> Esta es una aplicacion de Ruby on Rails 5 que permite a los usuarios crear su propio portafolio.

### Caracteristicas

- Motor de chat en tiempo real para los comentarios.
- Blog.
- Portafolio.
- Interface Arrastrar y Soltar
